Trang chủ

Admin quick

  1. Mở Google Sheet để quản lý menu: Mở Sheet
  2. Hướng dẫn nhanh: Cột headers: group,name,description,price,show,image (show viết TRUE để hiện)
  3. Đổi trạng thái hiện/ẩn: sửa cột show thành TRUE/FALSE
  4. Thêm ảnh: dán link ảnh vào cột image
  5. Publish CSV: không cần — site đọc trực tiếp CSV public.

Quản lý đánh giá

Reviews có thể được lấy từ Google Places API (cần API key). Nếu không dùng API, quản trị viên có thể chỉnh file data/reviews.json trong repo để thêm/ẩn đánh giá.
