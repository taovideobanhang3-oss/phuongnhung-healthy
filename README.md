PHUONGNHUNG-HEALTHY - Ready-to-deploy static site (uses Google Sheet for CMS)

Instructions to deploy to your repo 'phuongnhung-healthy':

1) In your GitHub repo (https://github.com/taovideobanhang3-oss/phuongnhung-healthy) click 'Add file' -> 'Upload files' and upload all files from this package.
2) Commit to 'main' branch.
3) Go to Settings -> Pages (or Settings -> Code and automation -> Pages), set Source to branch 'main' and folder '/' (root), Save.
4) Wait ~1-5 minutes; site will be available at: https://taovideobanhang3-oss.github.io/phuongnhung-healthy/

Important: Your Google Sheet must be shared 'Anyone with the link' -> Viewer. Current sheet link provided will work if sheet is viewable publicly.
